# Intent

## My Notes

I finally got it that an `action` has two parameters, `request` and `params`.
The `request` holds information about the form inputs.
The `params` holds information about the route variables.
Also, it was nice to learn that buttons inside a form are also input controls
and can therefore have names and values.
